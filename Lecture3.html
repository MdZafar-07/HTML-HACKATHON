📁 Lists in HTML

  • Unordered Lists

    An unordered list is used when the sequence of the items doesn't matter. It's created with the <ul> tag, and each item is defined with the <li>>(list item) tag. By default, items are marked with bullets.

    • Milk
    • Bread
    • Cheese
    
<ul>
    <li>Milk</li>
    <li>Bread</li>
    <li>Cheese</li>
</ul>

  • Ordered Lists

    An unordered list is used when the sequence of the items doesn't matter. It's created with the <ol> tag, and each item is defined with the <li>>(list item) tag. By default, items are marked with bullets.

    1. Milk
    2. Bread
    3. Cheese
    
<ol>
    <li>Milk</li>
    <li>Bread</li>
    <li>Cheese</li>
</ol>


📁 Nested Lists in HTML

A nested list is a list that is placed inside an item (<li>) of another list.
This creates a hierarchical or parent-child relationship between the lists.
The video compares this concept to nested loops in programming languages.


Example 1: Unordered List within an Unordered List

A main list of countries is created (<ul>).
Within the <li> for "India," a new <ul> is created for cities (Delhi, Mumbai, Goa).
This pattern is repeated for other countries like the USA and England.

  • India
    • Delhi
    • Mumbai
    • Goa
  • USA
    • New York
    • California
    • Texas
  • England
    • London
    • Manchester
    • Liverpool

Example 2: Mixed Nested Lists:

This example shows how to nest an ordered list (<ol≫) inside an unordered list (<ul>).
Within the "Delhi" list item, an ordered list is created for specific locations ("Dwarka" and "Vikaspuri")
Similarly, an ordered list for "Austin" and "Dallas" is nested under "Texas".

  • India
    • Delhi
      1. Dwarka
      2. Vikaspuri
    • Mumbai
  • USA
    • California
    • Texas
      1. Austin
      2. Dallas


📒 Tables in HTML

Tables are used to display data in a structured grid format of rows and columns.

Core Table Tags:

  • <table> : The primary tag that encloses all table content.
  • <tr> : (Table Row): Defines a single row in the table.
  • <td> : (Table Data): Defines a single cell within a row.

Note: To make the grid visible, the border="1" attribute is added to the <table> tag.

First Name Last Name Age
Amit Sharma 28
Priya Singh 32

Semantic Table Structure:

  • Table Headers (<th>):

    Used in place of <td> for header cells (e.g., "Name," "Role").They are semantically important for accessibility (screen readers) and SEO. Browsers typically render <th> text as bold and centered.

  • Professional Grouping Tags:

    <thead>: Groups the header rows (<tr> containing <th>) of the table.
    <tbody>: Groups the main data rows of the table.
    <tfoot>: Groups the footer rows, often used for summary data or totals.

Spanning Multiple Cells:

  • (Column Span):

    An attribute added to a <td> or <th> tag.It allows a single cell to stretch horizontally across multiple columns.Example: colspan="2" makes a cell occupy the space of two columns.

  • (Row Span):

    An attribute added to a <td> or <th> tag.It allows a single cell to stretch vertically across multiple rows.Example: rowspan="2" makes a cell occupy the space of two rows.
